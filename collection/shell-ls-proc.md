## Response Fields

- **items** (array): List of all items in /proc directory
- **count** (number): Total number of items

## Item Types

The /proc directory contains two main types of items:

### Process Directories (Numeric)
- **Numeric directories** (1, 2, 3, etc.) - Each represents a running process
- Process ID (PID) directories contain detailed process information
- Can be used to examine individual process status and statistics

### System Files (Alphanumeric)
- **cpuinfo** - CPU information
- **meminfo** - Memory information  
- **version** - Kernel version
- **mounts** - Mount points
- **devices** - Device information
- **interrupts** - Interrupt statistics
- **loadavg** - Load average
- **uptime** - System uptime
- **vmstat** - Virtual memory statistics
- **net/** - Network information directory
- **sys/** - System parameters directory

## Notes

- Shows all running processes as numeric directories (PIDs)
- Includes system information files for kernel and hardware details
- Useful for system monitoring and debugging
- Process directories can be examined for detailed process information
- System files provide real-time kernel and system statistics
- Total count varies based on number of running processes

## Related Commands

- `adbjson shell cat /proc/cpuinfo` - Get CPU information
- `adbjson shell cat /proc/meminfo` - Get memory information
- `adbjson shell cat /proc/version` - Get kernel version
- `adbjson shell ps -A` - List all processes with PIDs
