# wm-density Command

## Description
Get the screen density of the device.

## Command
```bash
adbjson wm-density
```

## Equivalent ADB Command
```bash
adb shell wm density
```

## Sample Output
```json
{
  "physical_density": "390"
}
```

## Flags
- `--pretty` - Pretty print JSON (default: true)
- `--compact` - Compact JSON output
