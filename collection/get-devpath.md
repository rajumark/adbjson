# get-devpath Command

## ADB Command
```bash
adb get-devpath
```

## Description
Get the device path of the connected device.

## Sample Output
```json
{
  "dev_path": "usb:20-1"
}
```

## adbjson Command
```bash
./adbjson get-devpath
```

## Flags
- `--pretty` - Pretty print JSON (default: true)
- `--compact` - Compact JSON output
