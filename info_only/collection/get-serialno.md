# get-serialno Command

## Description
Get the serial number of the connected device.

## Command
```bash
adbjson get-serialno
```

## Equivalent ADB Command
```bash
adb get-serialno
```

## Sample Output
```json
{
  "serial_no": "ZD222XW5RL"
}
```

## Flags
- `--pretty` - Pretty print JSON (default: true)
- `--compact` - Compact JSON output
