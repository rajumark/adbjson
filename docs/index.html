adbjson - Clean JSON from Android Debug Bridge

Transform raw ADB command output into structured, parseable JSON. Built for developers who need reliable device data in their automation workflows.

Zero configuration, instant results, and perfect integration with modern development pipelines.

Get Started View Documentation

Stop Parsing Text, Start Using Data

Raw ADB output breaks scripts and wastes development time. adbjson delivers consistent JSON that just works.

The Challenge

  • Inconsistent formatting across Android versions
  • Fragile regex parsing that breaks unexpectedly
  • No structured data for modern pipelines
  • Hours wasted on text processing instead of features

The Solution

  • Consistent JSON structure, every single time
  • Native parsing in any programming language
  • Stable schema that survives Android updates
  • Ready for production pipelines from day one

macOS

Install with brew 
brew tap adbjson/tap

Linux

Install with curl

Windows

Install with PowerShell 
irm https://get.adbjson.dev/install.ps1 | iex

Usage Examples

Start using adbjson immediately. It's a drop-in replacement for adb with JSON output.

List connected devices 
adbjson devices
Get battery information 
adbjson battery
Pipe to jq for filtering 
adbjson devices | jq '.devices[].serial'
Save to file 
adbjson shell dumpsys battery > battery.json
