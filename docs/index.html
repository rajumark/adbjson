adbjson - Transform ADB Output Into Clean JSON

Convert raw adb command outputs into structured JSON for scripts, tools, automation, and debugging workflows.

For developers, automation engineers, QA teams, and Android developers who need reliable, parseable device data.

Get Started View Documentation

The Problem ADB Output is Messy

Raw adb commands return inconsistent, hard-to-parse text that breaks scripts and wastes engineering time.

Why ADB Output is Painful

  • Inconsistent formatting across commands
  • Requires fragile regex parsing
  • Breaks when Android changes output format
  • No structured data for automation

How adbjson Solves It

  • Consistent JSON structure every time
  • Native parsing in any language
  • Stable schema across versions
  • Ready for pipelines and automation
Before: Raw ADB Output 
List of devices attached
emulator-5554   device
0123456789abcdef device

$ adb shell dumpsys battery
Current Battery Service state:
  AC powered: false
  USB powered: true
  Wireless powered: false
  Max charging current: 500000
  ...
After: Clean JSON 
{
  "devices": [
    {
      "serial": "emulator-5554",
      "status": "device"
    },
    {
      "serial": "0123456789abcdef",
      "status": "device"
    }
  ],
  "battery": {
    "acPowered": false,
    "usbPowered": true,
    "wirelessPowered": false,
    "maxChargingCurrent": 500000
  }
}

How It Works

Three simple steps to get structured Android device data.

1. Install

Download the binary or use your package manager. Single file, no dependencies.

2. Run Command

Execute adbjson with any supported adb command. It wraps adb automatically.

3. Get JSON

Receive clean, structured JSON output. Pipe to other tools or save to file.

Installation

Get up and running in seconds.

Quick Install

Choose your platform and install method. adbjson is a single binary with no runtime dependencies.

macOS

Install with curl 
curl -sSL https://get.adbjson.dev | sh
Install with brew 
brew tap adbjson/tap
brew install adbjson

Linux

Install with curl 
curl -sSL https://get.adbjson.dev | sh

Windows

Install with PowerShell 
irm https://get.adbjson.dev/install.ps1 | iex

Example Usage

Once installed, use adbjson just like adb but get JSON output.

List devices 
adbjson devices
Get battery info 
adbjson battery
Pipe to jq 
adbjson devices | jq '.devices[].serial'
